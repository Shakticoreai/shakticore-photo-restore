SHAKTICORE PHOTO RESTORE v1.0.0 - RELEASE PACKAGE
==================================================

This package contains TWO installation options:

OPTION 1: PORTABLE VERSION (RECOMMENDED)
-----------------------------------------
Location: portable\ folder

Advantages:
✅ No installation needed
✅ Models included - works immediately
✅ No admin rights required
✅ Run from USB drives
✅ Easy to update

Installation:
1. Copy the "portable" folder anywhere
2. Run shakticore-photo-restore.exe
3. Done!


OPTION 2: MSI INSTALLER
------------------------
File: ShaktiCore-Setup-v1.0.msi

Advantages:
✅ Traditional Windows installer
✅ Adds to Start Menu
✅ Easy uninstall via Control Panel

Installation:
1. Run ShaktiCore-Setup-v1.0.msi
2. Follow installer steps
3. IMPORTANT: After installation, you need to manually copy models
   See MSI_INSTALLATION_NOTES.txt for instructions

Note: MSI requires manual model installation due to a known issue.
This will be fixed in v1.1. For now, we recommend the portable version.


WHICH ONE TO CHOOSE?
--------------------
✅ PORTABLE: Best for most users - works immediately
⚠️ MSI: Only if you specifically need a traditional installer


SYSTEM REQUIREMENTS
-------------------
• Windows 10 or 11 (64-bit)
• 4GB RAM minimum
• 500MB free disk space
