SHAKTICORE PHOTO RESTORE v1.0.0
================================

What it does:
• Enhance photos with AI, 100% offline
• Fast mode: 1-2 seconds per photo  
• Quality mode: 7 seconds per photo (42x more detail)
• No internet, no uploads, no accounts

System Requirements:
• Windows 10 or 11 (64-bit)
• 4GB RAM minimum
• 500MB free disk space
• Optional: GPU for faster processing

Installation:
1. Extract this ZIP file
2. Run shakticore-photo-restore.exe
3. No installation needed - portable version!

How to Use:
1. Launch the app
2. Select Fast or Quality mode
3. Click "Select Folder & Process"
4. Choose a folder with images (JPG, PNG, BMP)
5. Wait for processing
6. Find enhanced images with "_enhanced" suffix

Performance:
• Tested: 6 images in 43.7 seconds
• Average: 7-8 seconds per image
• Quality improvement: 42x more detail (43KB → 1,812KB)
