SHAKTICORE PHOTO RESTORE v1.0 - FINAL RELEASE
==============================================

INSTALLATION OPTIONS
--------------------

OPTION 1: MSI INSTALLER (RECOMMENDED FOR MOST USERS)
File: ShaktiCore-PhotoRestore-v1.0.msi

✅ Professional Windows installer
✅ Adds to Start Menu
✅ Adds to Programs & Features
✅ Easy uninstall
✅ Models included in installation

Installation:
1. Double-click ShaktiCore-PhotoRestore-v1.0.msi
2. Follow installer steps
3. Launch from Start Menu
4. Done!

Note: Windows may show security warning. Click "More info" → "Run anyway"


OPTION 2: PORTABLE VERSION
Folder: portable\

✅ No installation needed
✅ Run from anywhere (USB drives, etc.)
✅ No admin rights required
✅ Models included

Installation:
1. Copy "portable" folder anywhere
2. Run shakticore-photo-restore.exe
3. Done!


FEATURES
--------
✅ AI Photo Enhancement (2x upscaling)
✅ Fast mode: 1-2 seconds per image
✅ Quality mode: 7-8 seconds per image
✅ 100% offline - no internet needed
✅ Drag & drop interface
✅ Quality protection (prevents over-processing)
✅ Batch processing


SYSTEM REQUIREMENTS
-------------------
• Windows 10 or 11 (64-bit)
• 4GB RAM minimum
• 500MB free disk space


HOW TO USE
----------
1. Launch the app
2. Select Fast or Quality mode
3. Drag & drop a folder OR click to browse
4. Wait for processing
5. Find enhanced images with "_enhanced" suffix
